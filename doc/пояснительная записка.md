## Функции
Разберём интерфейс программы, что за что отвечает.


### Функции "Файл"
Здесь присутствуют одни из важных функций для управления расположением проекта и с каким будем работать если не хотим перезапускать приложение.

Рассмотрим функции:
![file_functions.png](https://github.com/ArtemChikc/MacLearn/blob/main/doc/images/file_functions.png)

* **"Открыть"** -при нажатии открывается окно выбора папки проекта (то же самое что и при запуске программы), при выборе папки программа перезапускается уже с новым проектом который человек выбрал
* **"Сохранить"** -то же самое что и кнопка "Сохранить проект" во вкладке "Проект", сохраняет конфиг автоматизации установки датасета
* **"Сохранить как..."** -можно сохранить проект по новому пути, и если нужно сразу перезапустить программу по новому пути
* **"Выход"** -выход из программы, закрытие проекта


### Вкладка "Проект"
Рассмотрим интерфейс:
![project_tab.png](https://github.com/ArtemChikc/MacLearn/blob/main/doc/images/project_tab.png)

Первым делом нас встречает раздел "Классы объектов", он включает в себя поле классов, где мы можем добавлять классы и изменять их "подклассы" (информацию по которой будет поиск изображений). Далее идёт раздел "Настройки датасета".
Разберём функции каждого раздела

#### Функции классов:
* Вкл./выкл. класса из работы автодатасета (если данная часть уже была создана)
* Изменение названия класса, он будет нужен чтобы выделять с каким классом мы работаем, а так же в базе данных это выступает как id класса.
* Добавление класса через кнопку снизу поля
* Удаление класса
* Добавление подкласса

#### Функции подклассов:
Аналогичны, но есть изменения
* При нажатии на верхнее поле можно выбрать изображение, оно будет "примером" при поиске похожих изображений.
* Изменение текста для поиска данных изображений
* Удаление подкласса

#### Настройки датасета
Это выбор того что и как будет сделано в датасете
* **"Создание валидационных данных"** -выбор, скачивать ли дополнительные изображения, которые пойдут в отдельные папки для обучения модели "с учителем"
* **"Аугментированные изображение"** -выбор, создавать ли изображения с искажениями, в некоторых случаях это может улучшить распознование у модели в плохих условиях
* **"Аннотация"** -выбор, делать ли "аннотацию" изображений, то есть выделение объекта на изображениях, т.к. это достаточно затратная вещь из-за вычитания координат расположения объекта путём удаления фона, из-за этого выделена как выбор
* **"Формат аннотации"** -т.к. аннотация у каждого способа обучения разные сделал этот выбор, на данный момент пока-что будет только формат YOLO (самый лёгкий), и для этого будет отдельный удобно расширяемый модуль
* **"Изображений на класс"** -количество изображений которые будут устанавливаться на класс

## Пользование

### Советы по пользованию
* Пожалуйста, не изменяйте вручную содержимое проекта. Всё делается только через программу! Если так нужна ручная обработка то ждите следующие версии, т.к. сейчас даже автодатасет ещё не работает.
* Не забывайте сохранять настройки проекта после изменений

### Инструкция по первому запуску программы
Всё достаточно легко:
* 1) Запускаем программу и выбираем папку
* 2) Если будет такой вопрос
![project_not_find.png](https://github.com/ArtemChikc/MacLearn/blob/main/doc/images/project_not_find.png)
    Нажимаем "Да" если уверены что выбрали ту папку
* 3) Далее у нас открывается окно проекта и пользуемся функциями настраивая проект, по тому как я их описал
* 4) Если хотим запустить создание датасета у нас должны быть подготовлены данные, если они есть то переходим во вторую вкладку "Автодатасет", нажимаем "Запуск" и смотрим как создаётся наш датасет.

![autodataset_tab.png](https://github.com/ArtemChikc/MacLearn/blob/main/doc/images/autodataset_tab.png)
