### Структура данных датасетов
```
project1/
├── dataset/
│   ├── images/      # изображения для обучения и тренировки
│   │   ├── train/
│   │   └── val/
│   ├── labels/      # аннотация (может быть файл заместо папки, в будущем)
│   │   ├── train/
│   │   └── val/
│   └── dataset.yaml # конфиг для самой тренировки, будет базово заполняться судя по настройкам датасета
└── project.conf     # конфиг для моей программы
```
- Работа с файлами и самим проектом будет работать через специальный модуль
