# MacLearn документация

MacLearn — это программа для автоматического сбора и подготовки качественных датасетов из изображений, скачиваемых из интернета. Инструмент предназначен для ML-разработчиков, исследователей и всех, кто работает с компьютерным зрением.

## О проекте

MacLearn автоматизирует самый трудоёмкий этап разработки моделей машинного зрения — сбор и подготовку датасетов.
Преимущества:

### 🚀 Быстрый сбор данных
- Сбор тысяч изображений за считанные часы
- Создание полного датасета на 5000+ изображений в течение 1 дня
- Автоматическая загрузка и обработка изображений
- Экспортирование готовых датасетов в популярные форматы (YOLO, CSV и др.)

### 🎯 Качественная разметка
- Чёткая структуризация данных по классам
- Создание аннотации изображений
- Поддержка сложных запросов с множеством вариаций
- Гибкая система категоризации

### 🤖 Автоматическая аугментация
- Генерация +4 вариаций для каждого исходного изображения
- Разнообразие данных для улучшения обучения моделей

### ⚙️ Гибкие настройки обработки
- Возможность получения исходных изображений без обработки
- Контроль качества и разрешения

### 🖥️ Удобный интерфейс для управления датасетом
- Подготовка данных для датасета
- Создание и обработка данных с визуализацией
- Возможность ручной доработки датасетов

## Установка и первый запуск

1. **Запустите программу** и выберите папку для проекта.
2. Если появится окно с вопросом:
   
   ![project_not_find.png](https://github.com/b1t0nese/MacLearn/blob/main/doc/images/project_not_find.png?raw=true)
   
   Нажмите **«Да»**, если уверены, что выбрали нужную папку.
3. Откроется окно проекта. Настраивайте проект с помощью функций, описанных ниже.
4. Для запуска автоматического сбора перейдите во вкладку **«Автодатасет»** и нажмите **«Запуск»**.

## Использование программы

### Функции меню "Файл"

![file_functions.png](https://github.com/b1t0nese/MacLearn/blob/main/doc/images/file_functions.png?raw=true)

- **«Открыть»** — открыть окно выбора папки проекта. При выборе программа перезапускается с новым проектом.
- **«Сохранить»** — сохранить конфигурацию автоматизации (аналог кнопки «Сохранить проект» во вкладке «Проект»).
- **«Сохранить как...»** — сохранить проект по новому пути с возможностью сразу перезапустить программу.
- **«Экспорт датасета»** — экспортировать датасет в папку проекта согласно настройкам.
- **«Экспорт датасета как...»** — экспортировать датасет по указанному пользователем пути.
- **«Выход»** — закрыть программу.

---

### Вкладка "Проект"

![project_tab.png](https://github.com/b1t0nese/MacLearn/blob/main/doc/images/project_tab.png?raw=true)

#### Раздел «Классы объектов»

- **Навигация по классам** — выбор класса в табе сверху.
- **Поиск поисковых запросов** — можно вписать текст, по которому будет искаться поисковые запросы.
- **Вкл./выкл. класса** — включает или исключает класс из работы автодатасета.
- **Изменение названия класса** — название используется как идентификатор в базе данных.
- **Добавление класса** — кнопка снизу поля.
- **Удаление класса** — удалить выбранный класс.
- **Добавление подкласса** — создать подкласс для уточнения поиска.

##### Подклассы

- **Поле с изображением** — можно выбрать изображение-пример для поиска похожих.
- **Текст для поиска** — поисковый запрос для данного подкласса.
- **Удаление подкласса** — удалить подкласс.

#### Раздел «Настройки датасета»

- **«Создание валидационных данных»** — скачивать дополнительные изображения для валидации.
- **«Аугментированные изображения»** — создавать искажённые копии для улучшения распознавания.
- **«Аннотация»** — выделять объекты на изображениях (ресурсоёмкая операция).
- **«Формат аннотации»** — выбор формата разметки (на данный момент YOLO).
- **«Изображений на класс»** — количество изображений для каждого класса.
- **«Размер изображений»** — выбор размера исходных изображений.

### Вкладка "Датасет"

![dataset_tab.png](https://github.com/b1t0nese/MacLearn/blob/main/doc/images/dataset_tab.png?raw=true)

#### Основные функции
- **«Обновление данных»** — синхронизация с файловой системой.
- **«Экспорт датасета»** — выгрузить датасет по выбранному пути (по умолчанию — папка проекта).
- **«Выбор класса»** — переключение между классами в TabBar.
- **Навигация по разделам изображений** — выбор раздела в табе сверху.
- **Поиск изображений** — можно вписать текст, по которому будут искаться изображения.

#### Виджеты

##### Поле изображений «для типов»

![images_field_widget.png](https://github.com/b1t0nese/MacLearn/blob/main/doc/images/images_field_widget.png?raw=true)

- **«Экспорт изображений поля»** — выгрузить изображения текущего поля.
- **«Импорт изображений в поле»** — загрузить изображения из папки.
- **«Удаление изображений»** — удалить все изображения данного типа.
- **«+»** — добавить одно изображение в проект.

##### Карта изображения

![image_card.png](https://github.com/b1t0nese/MacLearn/blob/main/doc/images/image_card.png?raw=true)

- **Клик по изображению** — открыть окно ручного редактирования.
- **«Удалить»** — удалить конкретное изображение.

### Вкладка "Автодатасет"

![autodataset_tab.png](https://github.com/b1t0nese/MacLearn/blob/main/doc/images/autodataset_tab.png?raw=true)
![autodataset_tab_chrome.png](https://github.com/b1t0nese/MacLearn/blob/main/doc/images/autodataset_tab_chrome.png?raw=true)

#### Виджеты

- **Вкладки «Работа» и «chrome.exe»** — вторая вкладка позволяет поместить окно браузера внутрь программы.
- **«Выбранные классы»** — таблица со статусами установки изображений.
- **«Этапы работы»** — статусы этапов: установка, аннотация, аугментация, экспорт.
- **«Логи выполнения»** — подробный текстовый вывод.
- **Превью изображения** — последнее обработанное изображение и его путь.
- **Кнопка «Запуск/Стоп»** — запуск и остановка процесса.
- **Прогрессбар** — общий процент выполнения.

## Советы по использованию

- Не изменяйте вручную содержимое папки проекта. Все операции — только через программу.
- Сохраняйте настройки проекта после изменений.
- Используйте вкладку **«chrome.exe»**, чтобы окно браузера не мешалось на рабочем столе.
- Будьте внимательны: функции отмены (Ctrl+Z) в программе нет.
